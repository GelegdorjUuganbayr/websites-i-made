üîí Constants

  • A constant stores a value that never changes while the program runs.
  • Use constants to make programs easier to understand and maintain.
# Python example
PI = 3.14
radius = 5
area = 2 * PI * radius
print("Circumference:", area)

Use CAPITAL letters for constants in Python.

üßÆ Casting Between Data Types

Casting means converting one data type to another.

CommandPurposeExample
int()Convert to integerint("5") ‚Üí 5
float()Convert to real numberfloat("2.75") ‚Üí 2.75
str()Convert to stringstr(10) ‚Üí "10"
bool()Convert to Booleanbool(0) ‚Üí False, bool("hi") ‚Üí True

üíª Casting Examples in Python

x = "42"
print(int(x) + 8)     # converts string ‚Üí integer

num = 0
print(bool(num))      # outputs False

text = "False"
print(bool(text))     # outputs True (non-empty string)

Notice that even "False" (a string) becomes True because it‚Äôs not empty!
